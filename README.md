# Crocodile Dentist

간단한 악어 이빨 누르기 게임입니다. 위험한 이빨을 피해서 최대한 많은 안전한 이빨을 눌러보세요!

## 실행 방법

정적 HTML/JS 프로젝트이므로 별도의 빌드 과정 없이 실행할 수 있습니다.

1. 저장소를 클론하거나 압축을 해제합니다.
2. `public/index.html` 파일을 브라우저에서 직접 열거나 아래 명령으로 간단한 서버를 띄웁니다.

```bash
npm install --global serve   # 처음 한 번만 설치
serve public
```

서버가 실행되면 표시된 주소(기본값 http://localhost:3000)를 브라우저에서 열어 게임을 즐기면 됩니다.

> 💡 WebGL을 지원하는 최신 브라우저(Chrome, Edge, Firefox, Safari 등)가 필요합니다.

## 게임 방법

- `라운드 시작` 버튼을 누르면 악어가 임의의 위험 이빨을 설정합니다.
- 캔버스 위 악어의 이빨을 직접 클릭하여 안전한 이빨을 찾아보세요.
- 안전한 이빨을 누르면 초록색으로 잠기며 점수가 올라갑니다.
- 위험한 이빨을 누르면 즉시 패배하며 라운드가 종료됩니다.
- 모든 안전한 이빨을 찾으면 승리 메시지가 표시됩니다.

## 폴더 구조

- `public/index.html` – 메인 화면과 UI 레이아웃
- `public/css/styles.css` – 게임 전용 스타일
- `public/js/game.js` – 게임 로직 (라운드 관리, 점수 계산)
- `AGENTS.md` – 기여자 가이드
